# The Baobab Practice Website

Professional business website for The Baobab Practice - human-led organizational transformation and coaching.

## 🌳 About

The Baobab Practice partners with leaders and teams to navigate complex change through human-led, practice-based coaching. This website showcases our vision, mission, core values, services, and the Baobab Framework for adaptive transformation.

## 🛠️ Local Development

To run the website locally:

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
